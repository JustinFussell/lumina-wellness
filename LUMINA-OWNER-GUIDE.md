# Lumina Owner Guide

## Welcome

This is your studio command center.

### Key Areas

**Dashboard (`/Admin`)**
- See today's classes and real-time occupancy
- Quick links to attendance and schedule management

**Taking Attendance**
- Go to a class from the dashboard → mark who attended
- Simple checkboxes, one-click save

**Managing Schedule**
- View and cancel upcoming classes

### Member Experience
Members can:
- Browse the beautiful public schedule
- Book classes (uses credits)
- See their personal practice dashboard

### Demo Data
The system is currently seeded with realistic Rondebosch classes and a few sample members for testing.

### Next Steps (Recommended)
- Add your real instructors and class types in the database
- Connect Stripe for real payments
- Customize branding in the layout
